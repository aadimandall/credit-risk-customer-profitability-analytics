# Tableau Dashboard Plan

## Dashboard Title
Credit Risk & Customer Profitability Analytics

## Executive KPI Cards
1. Total portfolio exposure
2. Average predicted default probability
3. Total expected loss
4. Average risk-adjusted profitability
5. Customer count
6. Observed default rate

## Main Views
1. Default rate by credit limit segment
2. Expected loss by borrower segment
3. Risk-adjusted profitability by segment
4. Default probability distribution
5. Borrower risk-profitability quadrant
6. Key risk drivers
7. Recommended action by segment

## Suggested Filters
- Age group
- Credit limit segment
- Utilization segment
- Repayment behavior category
- Risk-profit segment
- Default flag

## Recommended Action Mapping

| Segment | Recommended Action |
|---|---|
| Low-risk / high-profit | Prioritize, grow, cross-sell |
| Low-risk / low-profit | Maintain efficiently |
| High-risk / high-profit | Monitor, reprice, set controls |
| High-risk / low-profit | Avoid, restrict, or decline |
